# Daniel Talks News Website

Static Netlify-ready website for Daniel Talks News, featuring Danny Williams as the Manchester-based host/spokesperson.

## Folder System

- `index.html`, `styles.css`, `script.js`: editable working site.
- `assets/`: source images used by the working site.
- `netlify-drop/`: clean folder that can be uploaded directly to Netlify.
- `releases/`: zipped deploy packages for drag-and-drop publishing.
- `docs/`: project notes, revision guidance, and folder-system notes.

## Deploy Now

Drag `releases/daniel-talks-news-netlify.zip` into Netlify Drop, or upload the `netlify-drop/` folder with `index.html` at the site root.
