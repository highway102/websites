# The Vehicle Longevity Black Book

This folder contains a first complete ebook draft and example graphics for:

**The Vehicle Longevity Black Book**  
**31 Overlooked Checks, Habits, Parts, and Model-Year Rules That Help Cars and Trucks Last Longer, Cost Less, and Stay Repairable**

## Files

- `index.html` - editable ebook source and browser preview.
- `styles.css` - ebook styling and print/PDF layout.
- `graphics/` - reusable SVG graphics for the ebook or sales pages.
- `Vehicle-Longevity-Black-Book.pdf` - 120+ page expanded generated PDF export after rendering.
- `Vehicle-Ownership-Secrets.html` - 100+ page Vehicle Ownership Secrets companion ebook source for the two-ebook sales bundle.
- `Vehicle-Ownership-Secrets.pdf` - 100+ page expanded generated companion PDF.
- `Vehicle-Longevity-Complete-Bundle.zip` - downloadable bundle containing both PDFs.
- `secrets.css` - companion manual styling and print layout.
- `truck-field-bundle/` - saved bundle access/offer section, linking both expanded PDFs.

## Graphics Included

- `inspection-map.svg` - 15-minute used vehicle inspection flow.
- `repairability-scorecard.svg` - repairability scoring graphic.
- `model-year-risk-matrix.svg` - redesign vs. proven-year risk matrix.
- `reset-calendar.svg` - 90-day used vehicle reset calendar.
- `towing-stress-chain.svg` - truck towing stress chain.

## Notes

The copy is written as educational material. Avoid marketing claims that promise guaranteed savings, emissions bypasses, illegal modifications, or universal mechanical outcomes.
